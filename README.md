# Weather Notification App

기상청 API를 기반으로 현재 위치 또는 선택한 지역의 날씨 정보를 확인하고, 설정한 시간에 날씨 알림을 받을 수 있는 안드로이드 날씨 앱입니다.

## 📌 프로젝트 소개

이 앱은 사용자가 원하는 지역의 날씨 정보를 쉽게 확인하고, 필요한 경우 알림을 통해 날씨 정보를 받아볼 수 있도록 만든 Android 프로젝트입니다.

기상청 단기예보 데이터를 활용하여 날씨 상태, 기온, 최고/최저 기온 등의 정보를 화면에 표시하며, 사용자가 설정한 시간에 알림을 받을 수 있도록 구현했습니다.

## 🛠 개발 환경

- Android Studio
- Kotlin
- Android SDK
- 기상청 단기예보 API

## ✨ 주요 기능

### 🌤 날씨 정보 조회

- 기상청 API를 이용한 단기 예보 조회
- 지역별 날씨 정보 표시
- 현재 기온 표시
- 시간대별 날씨 정보 표시

### 🔍 지역 검색

- 원하는 지역 검색 기능
- 선택한 지역 기준으로 날씨 데이터 조회
- 현재 위치 기반 날씨 조회

### ⭐ 즐겨찾기

- 자주 확인하는 지역을 즐겨찾기에 추가
- 즐겨찾기 지역 빠르게 날씨 확인
- 즐겨찾기 지역 알림 추가 가능

### 🔔 날씨 알림

- 사용자가 설정한 시간에 날씨 알림 제공

## 📱 화면 구성

- 메인 날씨 화면
- 지역 검색 화면
- 즐겨찾기 목록
- 알림 설정 화면
